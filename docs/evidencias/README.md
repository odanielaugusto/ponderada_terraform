# Evidencias

Esta pasta guarda os prints gerados a partir dos comandos executados no terminal.

| Arquivo | Conteudo |
| --- | --- |
| `01-terraform-version.png` | Versao local do Terraform |
| `02-terraform-fmt.png` | Execucao de `terraform fmt` |
| `03-terraform-init.png` | Inicializacao do provider AWS |
| `04-terraform-validate.png` | Validacao da configuracao |
| `05-terraform-plan-sem-credenciais.png` | Tentativa de `plan` bloqueada por falta de credenciais AWS |

Ao configurar credenciais AWS, gere novas evidencias para `plan`, `apply`, `output` e prints do console AWS mostrando EC2, VPC, subnet e security group criados.
